## NeoVim/Vim

### Requirements-

    - coc-nvim (installed globally using `sudo npm install -g coc.nvim`)
    - pyright (installed globally using `sudo npm install -g pyright`)
    - neovim package in npm (installed globally using `sudo npm install -g neovim`)

- clangd (installed using package manager, for Arch example- `sudo pacman -S clang`)

### Features-

    - Auto Code Completion
    - Linting
    - Automatic Install
    - Works for different languages like-
    - Javascript
    - React
    - Typescript
    - C/C++
    - HTML/CSS

    and a few more I don't remember.
